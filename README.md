# Numerical Methods Visualizer

A clean & minimal desktop app for these methods:
- Newton-Raphson
- Regula Falsi
- Gauss-Jacobi
- Gauss-Seidel

Built with Python + CustomTkinter, with a dark UI, readable iteration tables, rendered formulas, and a random input helper.

## Quick Start

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

## Build EXE

```bash
pyinstaller --onefile --windowed main.py
```
